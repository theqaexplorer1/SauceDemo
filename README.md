# Проект автотестирования SauceDemo

Автоматизированные тесты для приложения [SauceDemo](https://www.saucedemo.com) с использованием:
- Java + Selenium WebDriver
- TestNG
- Page Object Model

## 2. Запуск тестов

```bash
# Запустить все тесты
mvn test

# Запустить конкретный тест-класс
mvn test -Dtest=LoginTest

# Запустить конкретный тест-метод
mvn test -Dtest=LoginTest#checkLoginWithPositiveCreds
```
